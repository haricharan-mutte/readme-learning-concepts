# 📘 Complete Mutual Funds Guide (From Our Conversation)

## 🧠 1. What is a Mutual Fund?

A mutual fund is a pool of money collected from multiple investors and
managed by professionals (fund managers). This money is invested in: -
Stocks (Equity) - Bonds (Debt) - Other assets

------------------------------------------------------------------------

## 💸 2. What is SIP (Systematic Investment Plan)?

SIP is a way to invest a fixed amount every month into mutul fund. 

### How it works:

-   You invest monthly (e.g., ₹1000)
-   You buy units based on NAV
-   Market down → more units
-   Market up → fewer units

👉 This is called **Rupee Cost Averaging**

------------------------------------------------------------------------

## 📊 3. What is NAV (Net Asset Value)?

NAV = Price of one unit of a mutual fund

Formula: NAV = (Total Assets - Expenses) / Total Units

Example: - NAV = ₹50 - Investment = ₹1000 - Units = 20

------------------------------------------------------------------------

## 💰 4. What is Expense Ratio?

Annual fee charged by fund

Example: - Investment = ₹1,00,000 - Expense Ratio = 1% - Cost =
₹1,000/year

👉 Lower is better

------------------------------------------------------------------------

## 🚪 5. What is Exit Load?

Penalty for early withdrawal

Example: - Exit load = 1% within 1 year - Withdrawal = ₹1,10,000 -
Charge = ₹1,100

------------------------------------------------------------------------

## 🔒 6. Lock-in Period

Time during which you cannot withdraw money

-   ELSS funds → 3 years lock-in
-   Most funds → No lock-in

------------------------------------------------------------------------

## 🏢 7. What is AMC?

AMC (Asset Management Company) manages mutual funds

Examples: - SBI Mutual Fund - HDFC Mutual Fund - ICICI Prudential Mutual
Fund

------------------------------------------------------------------------

## 📂 8. Categories of Mutual Funds

### Equity Funds

-   High risk, high return
-   Long-term (5--10 years)

### Debt Funds

-   Low risk
-   Stable returns

### Hybrid Funds

-   Mix of equity + debt

### Solution-Oriented

-   Retirement, child plans
-   May have lock-in

### Others

-   Index funds, ETFs

------------------------------------------------------------------------

## 📁 9. Subcategories (Equity)

-   Large Cap → Stable companies
-   Mid Cap → Growing companies
-   Small Cap → High risk
-   Flexi Cap → Flexible allocation
-   Multi Cap → All sizes
-   ELSS → Tax saving
-   Sectoral → Single sector (risky)
-   Value/Contra → Strategy-based

------------------------------------------------------------------------

## 📉 10. What is Tracking Error?

Difference between index return and index fund return

-   Lower = better

------------------------------------------------------------------------

## 🧾 11. Fund Plan

-   Direct Plan → Lower cost (recommended)
-   Regular Plan → Higher cost

👉 Choose: Direct + Growth

------------------------------------------------------------------------

## 📈 12. SIP Growth Example

₹30,000/month SIP for 10 years: - Invested: ₹36,00,000 - Value:
\~₹70,00,000 (12% return)

------------------------------------------------------------------------

For ₹2L/month salary:

### SIP Amount:

-   ₹30K -- ₹50K recommended

-   40% Index Fund
-   40% Flexi Cap
-   20% Midcap

------------------------------------------------------------------------

## 🔍 14. How to Select a Fund

Check: 
-  1. Category(Index fund/)
-  2. 5/3-year returns(\>12%)
-  3. Expense ratio (low)(0-1%)
-  4. AUM (₹5K--50K Cr)
-  5. Portfolio diversification

------------------------------------------------------------------------

## Past Returns (but don’t misuse it)
```
Check:

3-year return
5-year return

👉 Good benchmark:

12–15% = good
15%+ = strong

⚠️ Don’t:

Chase 1-year high return (very common mistake)
```


## Expense Ratio (cost you pay)
- Index fund: ~0.1%–0.3% 👍
- Active fund: ~0.5%–1.5%

- 👉 Lower = better (especially)


## AUM (Assets Under Management)

```
👉 Meaning:
- Total money invested in that fund
- Good range: ₹5,000 Cr – ₹50,000 Cr

👉 Why?

Too small = risky
Too big = slow growth sometimes


# Fund Manager Track Record

👉 Ask:

How long managing?
Consistent performance?

Good managers stay:

5+ years in same fund

```

## Portfolio (MOST IMPORTANT 🔥)

```

Open “Holdings” section.

Check:

Top companies
Sector distribution

Example good signs:

Diversified sectors
Not too concentrated

```

## ⚠️ 15. Common Mistakes

-   Chasing high returns
-   Timing the market
-   Stopping SIP during market fall
-   Ignoring costs

------------------------------------------------------------------------

## 🎯 16. Key Rules

-   Start early
-   Stay consistent
-   Increase SIP yearly
-   Think long-term (10+ years)

------------------------------------------------------------------------

## 🧠 Final Understanding

-   You invest in units
-   Fund manager invests in companies
-   Market ups/downs help averaging
-   Wealth builds through compounding
